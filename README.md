# 🛒 Mi Tienda - Sistema de Gestión de Inventarios

Proyecto Full-stack profesional desarrollado con una arquitectura desacoplada, implementando las tecnologías más recientes de 2026.

## 🚀 Tecnologías y Frameworks

* **Backend:** [Laravel 12](https://laravel.com/) (PHP 8.2+) - Ubicado en `/backend/backend-laravel`
* **Frontend:** [Angular 21](https://angular.io/) - Ubicado en `/frontend/frontend-angular`
* **Base de Datos:** MySQL / MariaDB.
* **Estilos:** CSS3 Moderno (Glassmorphism & Grid).

---

## 🛠️ Requisitos Previos

* **XAMPP** (Apache, PHP 8.2+, MySQL).
* **Composer** y **Node.js**.

---

## 💻 Instalación y Configuración

### 1. Clonar el repositorio
```bash
git clone [https://github.com/TU_USUARIO/mi-tienda-2026.git](https://github.com/TU_USUARIO/mi-tienda-2026.git)
cd mi-tienda-2026

2. Configuración del Backend (Laravel)

cd backend/backend-laravel
composer install
cp .env.example .env
php artisan key:generate
# Configura tu .env y luego:
php artisan migrate
php artisan serve

3. Configuración del Frontend (Angular)

cd frontend/frontend-angular
npm install
ng serve
